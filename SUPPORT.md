# Support

## Getting Help

### GitHub Issues

For bugs, incorrect results, or broken scripts, please open a [GitHub Issue](https://github.com/spikora/SpikingTSF/issues) using the appropriate template. Be specific: include your environment, the command you ran, and the error output.

### GitHub Discussions

For general questions about the framework, model behavior, experimental design, or dataset usage, use [GitHub Discussions](https://github.com/spikora/SpikingTSF/discussions) if enabled, or open an issue with the `question` label.

### Email

For research collaboration inquiries or questions not suitable for public issues, contact the maintainer directly:

- **Jafar Bakhshaliyev** — [jafar.bakhshaliyev@gmail.com](mailto:jafar.bakhshaliyev@gmail.com)

### Response Time

This project is maintained on a best-effort basis alongside ongoing research. There is no guaranteed response time. For urgent issues, please be descriptive and concise in your initial message.

## Out of Scope

- Help with running models on hardware not yet tested (e.g., neuromorphic chips)
- Requests for custom implementations not aligned with the project roadmap
- General PyTorch or deep learning support unrelated to SpikingTSF
